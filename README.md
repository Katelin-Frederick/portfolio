A personal portfolio and blog built with Next.js, showcasing my experience, skills, and projects as a Full-Stack Engineer.

## Features

- **Single-page portfolio** with smooth-scrolling sections: Landing, About, Experience, Skills, Projects, and Contact
- **MDX blog** with syntax highlighting, pagination, and frontmatter-driven metadata
- **Animated UI** powered by Motion (Framer Motion)
- **Contact form** with Zod validation and Netlify Forms integration
- **Responsive design** built with Tailwind CSS v4 and shadcn/ui components

## Tech Stack

| Category | Technologies |
|---|---|
| Framework | Next.js 16, React 19, TypeScript |
| Styling | Tailwind CSS v4, shadcn/ui, Radix UI |
| Animations | Motion (Framer Motion) |
| Forms | React Hook Form, Zod |
| Blog/Content | MDX (next-mdx-remote), gray-matter, rehype-pretty-code |
| Deployment | Netlify |

## Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/Katelin-Frederick/portfolio.git
cd portfolio
```

2. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
